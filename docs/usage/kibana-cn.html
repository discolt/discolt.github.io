Kibana

  • 接入方法
  • 启动

可以使用Kibana的便捷的查询和分析存储在云端 Elastisearch 中的数据。除了柱状图、线状图、饼图、环形图等经典可视化功能外，还拥有地理位置分析、数据图谱分析、时序数据分析等高级功能。

接入方法#

除了使用 ZSearch 自带的 Kibana 中文版外， 您也可以下载官方 Kibana 到本地连接云端 ZSearch。

  1. 进入 Kibana 官网 下载 Kibana OSS 6.6.2 版本。

  2. 将下载的安装压缩包解压后，进入 Kibana 目录。

  3. 打开配置文件 config/kibana.yaml， 对照控制台的应用信息修改配置文件。 

# 示例配置
elasticsearch.url: "http://zsearch.cloud.alipay.com:9999"
elasticsearch.customHeaders: {"kibana-version":"6.6.2", "Authorization": "Basic xxxxxxx"}

  • elasticsearch.url 对应 API Server地址.

  • Authorization 是用户名和密码组合的Base64值。

Base64值的获取可以前往 Base64 加密工具网站。

  1. 在左侧框中以 用户名+":"+密码 的格式输入待加密信息。

  2. 点击 Base64 加密，即可获得一个加密值。

  3. 返回配置文件，修改 elasticsearch.customHeaders 参数里的 Authorization 值为该加密值。

启动#

使用 bin/kibana 命令启动 Kibana，访问 http://127.0.0.1:5601，即可进入 Kibana 界面。
