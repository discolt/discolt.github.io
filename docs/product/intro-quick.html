快速开始

  • 1. 创建应用
  • 2. Elasticsearch API
  • 3. Kibana 示例
  • 4. JAVA 客户端示例

快速开始帮忙您从本机电脑连接到 ZSearch， 快速体验云端搜索功能特性。

主要步骤#

  • 创建应用获取证书 - BasicAuth 用户名密码

  • 通过证书访问云端 Elasticsearch API

  • 一个简单的数据索引和检索实例

1. 创建应用#

进入 控制台 创建应用，我们以标准应用为例。

应用创建完成后，可以看到平台为该应用分配的 API服务器地址 和 用户名密码 ，为了方便使用，ZSearch采用了最为简单的 HTTP Basic Auth 身份验证，后端服务由强大的蚂蚁安全团队来保障。

2. Elasticsearch API#

打开终端控制台 通过颁发的授权信息访问服务

curl -u greeting_r3:99264a9263b5e http://zsearch.cloud.alipay.com:9999

curl 是一个非常方便的 Restful 调试工具， macOS、ubuntu中已内置，Windows中需要另行安装。

3. Kibana 示例#

进入 Kibana 控制台 Dev Tools ，体验最简单的索引检索功能。

  • 建立索引

PUT /employee

  • 插入数据

POST /employee/_doc
{
    "first_name" :  "Douglas",
    "last_name" :   "Fir",
    "age" :         35,
    "about":        "I like to build cabinets",
    "interests":  [ "forestry" ]
}

  • 检索数据

POST /employee/_search
